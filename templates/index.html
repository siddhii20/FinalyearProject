About

Intrusion detection is defined as the process of monitoring the events occurring in a computer system or network. An IDS detects all kinds of malicious network traffic that cannot be detected by a firewall such as network attacks against vulnerable services, data driven attacks on applications,unauthorised logins and access to sensitive files. IDS can not only separate normal network data from attack data, but also classify attack types. Therefore an IDS must classify a given set of network packets as normal or attack.
